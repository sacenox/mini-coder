👾 mini-coder

Small. Fast. Gets out of your way.

mc

⚡ Quick Install

bun add -g mini-coder
# or: npm install -g mini-coder

# Set your provider key
export OPENCODE_API_KEY=your-key

# Launch
mc

🔍 Smart Code Tools

glob, grep, read, replace, insert, create, shell, and subagent — every tool earns its spot.

🌐 Web Search

Built-in webSearch and webContent when EXA_API_KEY is set. Knowledge at your fingertips.

🤖 Multi-Provider

Auto-discovers Zen, Anthropic, OpenAI, Gemini, or local Ollama. No lock-in.

💾 Session Memory

SQLite-backed sessions. Resume where you left off with -c or pick a session with -r.

⚙️ Custom Everything

Commands, agents, skills, and hooks via .agents/ — follows community conventions.

🎨 Terminal Native

16 ANSI colors only — inherits your terminal theme. Dark mode, light mode, whatever you rock.

GitHub 📖 Documentation

Built with ❤️ and a healthy obsession with terminal aesthetics.
